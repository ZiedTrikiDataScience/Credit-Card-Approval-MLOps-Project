# Flask Deployment in Production :
run waitress-serve --listen=0.0.0.0:5001 app:app in the terminal 


# Docker Commands :

#### Building the docker image : 

docker build -t credit-card-approval-prediction:v1 .

#### Running the docker image : 

docker run -it --rm -p 5001:5001 credit-card-approval-prediction:v1


##### Running with Debugging  :
docker run -it --rm -p 5001:5001 credit-card-approval-prediction:v1 /bin/sh
